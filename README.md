# cybersecurity-project-101
This project demonstrates a Purple Team workflow: simulating an SSH brute-force attack using Kali Linux and Hydra, then detecting it in Splunk. I configured log ingestion from a Linux VM to build a real-time dashboard and alerting system. It showcases my skills in SIEM management, log analysis, and defending against unauthorized access.
